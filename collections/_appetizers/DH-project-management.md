1. A great [summary of project management methods written for researchers and academics](https://thestrugglingscientists.com/project-management/)  
2. [Clear explanation of the Kanban system](https://www.geeksforgeeks.org/software-engineering/what-is-the-kanban-system-and-how-does-it-work/) for project management  
3. [Trello project management tool](https://trello.com/): Free, with paid options for more project boards and software integrations.  
4. [Clickup project management tool](https://clickup.com/): Free for individuals, paid options for teams.  
5. [Freedcamp task management tool](https://freedcamp.com/): Core features are free, paid options for add-on features.
